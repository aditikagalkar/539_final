Capital One: Feedback to the Future

Senior Capstone Project (University of Texas at Dallas)

Duration

August 2022 – December 2022

Tech Stack

TypeScript, HTML, CSS, LIT Element, Figma

Role

Frontend Developer

Overview

Feedback of the Future is a project hosted by Capital One with the goal of providing a central place for employees to view customer feedback and assign problems to the necessary teams. In order to do so effectively, our team will create a fast and user-friendly application that allows easy viewing and editing of hundreds of data points simultaneously.

Wireframes

My Experience

I conceived a design for the UI of the Issue Page on Figma after researching and drawing inspiration from popular designs implemented on popular websites. Then, I created a new Typescript project utilizing Lit and set up the file structure for the frontend team’s project. For implementation, I worked on the individual Issue Page(s) which displays additional information on a customer and their review/issue that is not seen in the Landing Page. Such information included the customer’s full review, the customer’s contact information, and updates regarding how Capital One is handling the issue. These updates are seen through buttons for the issue status and team assigned, as well as a customer profile form and comment section. Added functionality to these features through integration with the backend to make API calls.
